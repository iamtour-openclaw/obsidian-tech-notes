# Obsidian 科技新闻笔记库

## 📖 项目简介

这是一个公开的 Obsidian 笔记库，收录每日重要的科技新闻、分析报告和趋势觀察。笔记按照日期组织，每天一个目录。

## 📅 笔记组织方式

### 按日期组织
- 每天创建一个新的日期目录（YYYY-MM-DD）
- 當日所有笔记都放在對應的日期目录中
- 便於按时间查找和归档

### 笔记类型
1. **新闻索引** (`📰 科技新闻索引.md`) - 當日新闻摘要和导航
2. **单篇新闻** - 具体新闻事件的详细记录
3. **分析报告** - 深度分析和趋势觀察
4. **專题研究** - 特定主题的深入研究

## 🚀 使用说明

### 1. 使用 Obsidian 打开
```bash
# 安装 Obsidian
# 下载地址：https://obsidian.md

# 打开笔记库
1. 打开 Obsidian
2. 点击"打开文件夹"
3. 选择本目录
4. 开始浏览和编辑笔记
```

### 2. 主要功能
- **双向链接**：笔记之间的智能链接
- **图谱视图**：可视化笔记关系网络
- **标签系统**：灵活的标签分类
- **搜索功能**：全文搜索和过滤

### 3. 浏览笔记
- **按日期浏览**：进入對應的日期目录
- **使用索引**：查看 `📰 科技新闻索引.md` 獲取當日新闻概覽
- **图谱视图**：在 Obsidian 中查看笔记关系图

## 🔧 开发指南

### 添加新笔记
1. 创建對應日期的目录（如果不存在）
2. 在日期目录中创建新的 Markdown 文件
3. 使用标准的笔记格式
4. 更新當日的 `📰 科技新闻索引.md`

### 笔记格式规范
```markdown
# 标题

**日期**: YYYY-MM-DD
**标签**: #标签1 #标签2

## 内容

正文内容...

## 相关链接

- [[相关笔记1]]
- [[相关笔记2]]
```

### 维护索引
- 每日更新當日的新闻索引
- 确保所有笔记都有正确的链接
- 定期清理和归档舊笔记
