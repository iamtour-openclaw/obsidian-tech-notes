## 📅 筆記組織方式

### 按日期組織
- 每天創建一個新的日期目錄（YYYY-MM-DD）
- 當日所有筆記都放在對應的日期目錄中
- 便於按時間查找和歸檔

### 筆記類型
1. **新聞索引** (`📰 科技新聞索引.md`) - 當日新聞摘要和導航
2. **單篇新聞** - 具體新聞事件的詳細記錄
3. **分析報告** - 深度分析和趨勢觀察
4. **專題研究** - 特定主題的深入研究

## 🚀 使用說明

### 1. 使用 Obsidian 打開
```bash
# 安裝 Obsidian
# 下載地址：https://obsidian.md

# 打開筆記庫
1. 打開 Obsidian
2. 點擊"打開文件夾"
3. 選擇本目錄
4. 開始瀏覽和編輯筆記
```

### 2. 主要功能
- **雙向鏈接**：筆記之間的智能鏈接
- **圖譜視圖**：可視化筆記關係網絡
- **標籤系統**：靈活的標籤分類
- **搜索功能**：全文搜索和過濾

### 3. 瀏覽筆記
- **按日期瀏覽**：進入對應的日期目錄
- **使用索引**：查看 `📰 科技新聞索引.md` 獲取當日新聞概覽
- **圖譜視圖**：在 Obsidian 中查看筆記關係圖

## 🔧 開發指南

### 添加新筆記
1. 創建對應日期的目錄（如果不存在）
2. 在日期目錄中創建新的 Markdown 文件
3. 使用標準的筆記格式
4. 更新當日的 `📰 科技新聞索引.md`

### 筆記格式規範
```markdown

## 內容

正文內容...

## 相關鏈接

- [[相關筆記2]]
```
